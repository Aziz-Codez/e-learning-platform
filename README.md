# E-Learning Platform

A modern e-learning platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Authentication system
- Course catalog
- Dark mode support
- Modern UI components

## Tech Stack

- Next.js 13+ (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI
